Расчётно-графическая работа по дисциплине

## 📌 Описание проекта

В рамках работы разработано Web-приложение на базе **Streamlit** для:

- анализа датасета `smokeEDA.csv`
- визуализации зависимостей признаков
- инференса (предсказаний) шести моделей машинного обучения
- пакетного предсказания по загружаемому CSV-файлу

Целевая задача — **бинарная классификация**:  
предсказание срабатывания пожарной сигнализации (`Fire Alarm = 0/1`) на основе показаний датчиков дыма.

## 🧠 Используемые модели

В проекте реализован инференс шести лучших моделей (по метрике **F1-score**).

### ✅ ML1 — Классическая модель
**Logistic Regression**  
Сохранена через `joblib` / `pickle`.

### ✅ ML2 — Ансамбль (бустинг)
**GradientBoostingClassifier**  
Сохранена через `joblib`.

### ✅ ML3 — Продвинутый градиентный бустинг
**CatBoostClassifier**  
Сохранена встроенным методом `.save_model()`  
Формат файла: `.cbm`.

### ✅ ML4 — Ансамбль (бэггинг)
**BaggingClassifier**  
Сохранена через `joblib`.

### ✅ ML5 — Ансамбль (стэкинг)
**StackingClassifier**  
Сохранена через `joblib`.

### ✅ ML6 — Глубокая нейронная сеть
Полносвязная Dense Neural Network (TensorFlow).  
Сохраняется встроенным методом:

```python
model.save("models/ML6_DenseNN.keras")
```

или

```python
model.save("models/ML6_DenseNN")
```

Поддержка ML6 уже предусмотрена в коде приложения (закомментирована до добавления модели).

## 🌐 Структура Web-приложения

Приложение состоит из четырёх страниц.

---

### 📄 Страница 1 — О разработчике

- ФИО
- Учебная группа
- Цветная фотография
- Тема РГР

---

### 📊 Страница 2 — Информация о датасете

- Первые строки таблицы
- Размерность датасета
- Типы данных
- Проверка пропусков
- Распределение целевого признака
- Описательная статистика

---

### 📈 Страница 3 — Визуализации (EDA)

Реализованы минимум 4 вида визуализаций:

1. **Heatmap корреляционной матрицы**
2. **Boxplot признака по классам**
3. **Scatterplot двух признаков с раскраской по классу**
4. **Гистограмма распределения признака**

Используемые библиотеки:
- Matplotlib
- Seaborn

---

### 🤖 Страница 4 — Инференс моделей

Поддерживаются два режима работы.

#### ✅ Ручной ввод одного объекта
- Ввод всех признаков
- Настраиваемый порог классификации
- Вывод вероятности класса 1
- Интерпретация результата:
  - 0 — сигнализация не сработает
  - 1 — сигнализация сработает

#### ✅ Batch-предсказание
- Загрузка CSV-файла
- Проверка структуры данных
- Добавление колонок:
  - `proba_fire_alarm_1`
  - `pred_fire_alarm`
- Возможность скачать результат

## 📊 Датасет

Файл: `smokeEDA.csv`

Тип задачи: **Бинарная классификация**

Целевая переменная:
```
Fire Alarm
```

Значения:
- 0 — сигнализация не срабатывает
- 1 — сигнализация срабатывает

## 📈 Метрика оценки моделей

Для выбора лучших моделей использовалась метрика:

```

```

Причина выбора:
- задача бинарной классификации
- возможный дисбаланс классов

## ⚙ Установка и запуск

### 1️⃣ Установка зависимостей

```bash
pip install streamlit pandas numpy scikit-learn catboost matplotlib seaborn joblib tensorflow
```

Если нейронная сеть ещё не используется, `tensorflow` можно не устанавливать.

---

### 2️⃣ Запуск приложения

```bash
streamlit run app.py
```

После запуска откроется веб-интерфейс дашборда.
