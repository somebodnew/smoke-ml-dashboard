# Дашборд для классификации возгораний (Streamlit)

Этот документ содержит полное описание и код веб-приложения (дашборда), разработанного в рамках расчетно-графической работы (РГР) по дисциплине «Машинное обучение и большие данные».

- **Датасет:** `smokeEDA.csv`
- **Задача:** Бинарная классификация
- **Целевой признак:** `Fire Alarm` (1 — есть возгорание, 0 — нет)
- **Технологии:** Python, Streamlit, Pandas, Scikit-learn, CatBoost, TensorFlow.

---

## 1. Функционал приложения

Приложение состоит из четырех основных разделов, переключаемых в боковой панели:

1.  **О разработчике**: Краткая информация об авторе проекта.
2.  **О датасете**: Обзор набора данных `smokeEDA.csv`, включая:
    - Первые 20 строк данных.
    - Размерность, типы данных и наличие пропусков.
    - Описательная статистика и распределение целевого класса.
3.  **Визуализации**: Четыре интерактивных графика для анализа данных:
    - Тепловая карта корреляций признаков.
    - Boxplot для анализа распределения признака в разрезе классов.
    - Scatterplot для изучения взаимосвязи двух признаков.
    - Гистограмма распределения для выбранного признака.
4.  **Инференс**: Основная страница для получения предсказаний от моделей.
    - **Выбор модели**: Возможность выбрать одну из 6 обученных моделей.
    - **Ручной ввод**: Форма для ввода значений признаков для одного объекта.
    - **Загрузка CSV**: Возможность загрузить файл для получения предсказаний для целой группы объектов (batch-инференс).

## 3. Установка и запуск

### Шаг 1: Подготовка окружения

Рекомендуется использовать виртуальное окружение для изоляции зависимостей.

```bash
# Создание виртуального окружения
python -m venv venv

# Активация (Windows)
.\venv\Scripts\activate

# Активация (macOS/Linux)
source venv/bin/activate
```

### Шаг 2: Установка зависимостей

Файл `requirements.txt` находится в корне проекта, убедитесь в его наличии и выполните установку:

```bash
pip install -r requirements.txt
```

### Шаг 3: Запуск приложения

Убедитесь, что все файлы (`app.py`, `smokeEDA.csv`, папка `models`) находятся в правильных директориях.

Запустите приложение командой:

```bash
streamlit run app.py
```

В вашем браузере автоматически откроется новая вкладка с работающим дашбордом.
